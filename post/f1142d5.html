环境准备

  1. JDK，版本至少需要 JDK8
  2. MySQL，版本至少需要 5.6
  3. Tomcat，版本至少需要 9
  4. Maven，下载解压后设置 bin 目录到 PATH 环境变量中
    在centos7下安装的

    程序安装

    一、JDK 安装

1. 安装 wget

2. 下载，输入指令

3. 执行 npm 安装包，等待安装完成

4. 验证安装是否成功

二、Tomcat 安装

  1. 下载，切换至任意目录执行

  2. 解压 ，执行以下命令

tar -zxvf apache-tomcat-9.0.0.M18.tar.gz

  1. 移动至想放置的目录

mv apache-tomcat-9.0.0.M18 /usr/local/

  1. 配置自启动，切换至 Tomcat 的 bin 目录执行 vi setenv.sh，并按 i 进入编辑模式，拷贝以下代码粘贴

#add tomcat pid
CATALINA_PID=”$CATALINA_BASE/tomcat.pid”

#add java opts
JAVA_OPTS=”-server -XX:PermSize=256M -XX:MaxPermSize=1024m -Xms512M -Xmx1024M -XX:MaxNewSize=256m”

然后输入 wq 保存退出，最后执行 chmod +x setenv.sh，确保文件 setenv.sh 可执行。

5. 配置 service 执行 cd /usr/lib/systemd/system 切换至目录 并执行 vi tomcat.service（也可以直接 vi /usr/lib/systemd/system/tomcat.service）, 拷贝以下代码粘贴（注意将第 6 和 7 行的 Tomcat 路径 / usr/local/apache-tomcat-9.0.0.M18 换成你自己的 Tomcat 的绝对路径）

[Unit]Description=TomcatAfter=syslog.target network.target remote-fs.target nss-lookup.target[Service]Type=forkingPIDFile=/usr/local/apache-tomcat-9.0.0.M18/tomcat.pidExecStart=/usr/local/apache-tomcat-9.0.0.M18/bin/startup.shExecReload=/bin/kill -s HUP $MAINPIDExecStop=/bin/kill -s QUIT $MAINPIDPrivateTmp=true[Install]WantedBy=multi-user.target

之后按 esc 键，在按 shift+：（左下角出现会出现 “：”），然后输入 wq 保存退出。执行以下指令将 tomcat.service 添加开机至开机启动

#配置开机启动
systemctl enable tomcat

#启动
systemctl start tomcat

#停止
systemctl stop tomcat

#重启
systemctl restart tomcat

三、ntos7+maven 安装
1、下载安装包

2、解压安装包

tar -xvf apache-maven-3.5.4-bin.tar.gz

3、移动至 / usr/local 下

mv apache-maven-3.5.4 /usr/local

4、配置环境变量，编辑 / etc/profile 文件，添加如下代码

AVEN_HOME=/usr/local/apache-maven-3.5.4
export MAVEN_HOME
export PATH=${PATH}:${MAVEN_HOME}/bin

5、保存文件，并运行如下命令使环境变量生效

ource /etc/profile

6、输入如下命令，如果能看到 Maven 相关版本信息，则说明 Maven 已经安装成功

mvn -v
