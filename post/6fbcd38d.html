OSFMount群晖镜像修改软件

发表于 2019-06-30 | 更新于 2019-09-19 | 分类于 默认分类 
OSFMount 是一个支持光盘镜像和内存虚拟盘创建装载实用小工具。OSFMount这个软件特色在于，既可以加载光盘镜像，也可以创建内存虚拟盘。

在创建黑群晖引导盘的时候我们需要把img文件写入U盘后才修改VID和PID但用这个软件可以提前修改img文件，然后保存后直接写入U盘。

1×01打开软件

1×02 主界面

1×02参照设置 特别注意 Read-only drvie(只读模式不能打勾)

确认无误 点击确认

这时候多出一个虚拟驱动器

进入驱动器目录/grub/找到grub.cfg文件使用Notepad++（推荐）

set vid=0x0DD8     //修改 0DD8 为你U盘的VID 保留前面的0x
set pid=0x0C00     //修改 0C00 为你U盘的PID 保留前面的0x
set sn=C7LWN09761    //此处为序列号 如你有的洗白需求的话可修改
set mac1=00113207DCBD //此处为网卡MAC修改为和你NAS一致的MAC 洗白的话需要001132开头 需要该硬件MAC
                       顺便说一句如果有远程唤醒的需求的话也得改mac地址和真实实体网卡mac地址一样

然后保存文件。在 OSFMount软件主界面点击卸载全部并退出

就这样就修改完成了然后写入U盘就可以了。
