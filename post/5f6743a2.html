在Debian上安装OpenMediaVault开源NAS系统

发表于 2019-02-14 更新于 2019-09-19

序言

先安装debian9系统任何再修改源然后再安装OpenMediaVault

注意事项

安装语言务必选择English
安装服务的时候择安装SSH

内容

1.添加源

2.安装OMV源key

3. 升级系统并安装OMV

apt update
apt install postfix openmediavault

3.1 初始化

omv-initsystem

4. 添加OMVExtras插件库

wget -O - http://omv-extras.org/install | bash

ps后续安装体验

我的环境在虚拟机里安装的，这种用网络的安装方式非常依赖网络环境加上这个NAS系统相关文件都是国外的，所以网络环境非常糟糕的话非常容易在下载openmediavault服务器上的东西时候把ssh弹飞，就是突然短线，或者卡死，我是在虚拟机终端界面手打命令才成功的。所以这种安装方式只是在官方的ISO镜像无法安的情况下做一个备用的选择。特别是比较老的服务器阵列卡无法支持，在安装的时候没法识别到硬盘，而debian安装的时候又可以识别正常安装。而我在r410服务器上是无法安装成功的，而debian也是无法安装成功原因都是无法识别硬盘或者安装中途中报错卡死。而pve虚拟化平台就可以安装成功按道理都是基于debain这个系统平台，

安装后

debian原版系统安装后只允许普通用户登录安装nas系统后普通用户不允许登录只允许root用户登录。
